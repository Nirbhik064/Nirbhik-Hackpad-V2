# Nirbhik-Hackpad-V2.
This is a hackpad made by me in 40+hours as a beginner. I then rearranged almost everything that it contained from its PCB design to its case. All the required files have been posted.
Also, while adding this repo, its 2:50 am. I have been up all night to make it complete, if anything is missing, please understand that I'll provide it with everything.

This macropad is used in every way possible on how a 4x4 keyboard can be used. As it also has an oled screen, it can be used ass a streamdeck and well, to edit videos, make more PCBs and much more. You just have to program it.

I made this project as a starting point into the world of pcb building and I was interested.

The parts that are used in this project are;
1. 1x Seeed XIAO RP2040 microcontroller.
2. 16x through-hole 1N4148 Diodes.
3. 16x MX-Style switches
4. 1x 0.91 inch OLED display
5. 16x white blank DSA keycaps
6. 4x M3x16mm screws
7. 4x M3x5mx4mm heatset inserts.
